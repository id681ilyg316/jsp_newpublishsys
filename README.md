## 本项目实现的最终作用是基于JSP在线新闻管理系统包含前后台
## 分为2个角色
### 第1个角色为管理员角色，实现了如下功能：
 - 修改个人资料
 - 修改密码
 - 新闻管理
 - 用户管理
 - 管理员登录
### 第2个角色为用户角色，实现了如下功能：
 - 按分类查看
